为什么移动网站建设越来越受市场的追捧？

织梦科技•2015-04-29•经验之谈•

在美国有这样一组数据，电话在1878年进入市场，但它用了71年的时间才让美国一半的家庭拥有了一部电话。从1907年开始的50年里，企业主们才逐步考虑是否要为自己的公司安装一部电话。

相比之下，自2007年智能手机崛起的时候，新一代的企业主们大概只有50个月的时间考虑是否要搭建一个移动端网站。科技的发展速度不禁让人惊叹，现在大部分的企业都已经完成了自己的移动网站建设，不过遗憾的是只有6%的网站是真正对移动端友好的。

为什么移动网站建设如此之快地就受到了市场的追捧？

1. 移动流量已超过PC流量，成为主流

2013年谷歌宣布移动端流量超过PC，2014年百度三季度财报显示，其移动流量全面超PC，比预期提前一个季度完成。阿里巴巴、腾讯、京东等互联网巨头也为了布局和抢占移动端用户，展开了激烈争夺。

今天的美国，已经有75%的手机用户在使用智能手机。2013年6月底，中国网民数达到5.9亿，其中手机网民达4.64亿，占网民总数的78.5%。这标志着移动端网站建设更加贴近客户。

2. 移动搜索用户有较高的购买意图

有数据统计，超过50%的消费者使用智能手机来搜索并决定他们要买什么，以及在哪里买。最常见的一个微妙动作，就是在搜索完成后，直接打一个电话进行咨询。

事实上，在2013年移动搜索就为企业创造了300亿次的销售电话呼入，而这一数字一直在增长，根据最近的一份报告显示，到2018年，移动搜索将为企业带来至少730亿个电话呼入。

另一项研究表明，有78%的消费者习惯使用智能手机来搜索当地的商家，并形成最终购买。

3. 移动网站建设可帮助用户更快地采取行动

同样的研究还显示，那些习惯浏览移动网站并最终决定购买的消费者，有四分之三 ( 76% ) 的购买行为是发生在浏览当天，其中有63%发生在几小时内。

4. 百度移动搜索推广要求必须使用移动网站

在国内，百度仍然占据着搜索市场的头把交椅，今天2月份，百度发布2014年第四季度财报，显示其移动营收在百度第四季度总营收中所占比例为42%，高于上一季度的36%。百度董事长兼首席执行官李彦宏表示：“在12月份，来自于移动端的搜索营收首次超过了PC端。”

目前，百度要求在其搜索引擎中做推广的商家，必须首先完成自己的移动网站建设，才能进行搜索推广，移动端优先显示移动网站的搜索结果，这也使得网站建设市场中，对移动网站建设的追捧热度更加高涨。
