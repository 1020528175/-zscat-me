Shopify，一枚简洁清新的国外电子商务平台网站

织梦科技•2015-04-29•经验之谈•

Shopify ( shopify.com ) 是国外的一家专门帮助新进入电商领域创业者们的多功能电商平台网站，在某程度上也是电子商务巨头 Amazon ( 亚马逊 ) 和 eBay ( 易贝 ) 的竞争者。

第一眼看到 Shopify 这个网站的时候，充满了小清新的感觉，采用绿色调的色彩搭配，使网站充满了活力。其网站制作采用了国外已经流行的 Html5 技术，并且实现了响应式布局，无论我们在PC电脑、笔记本电脑、Pad，还是手机上看，都可以获得完美的视觉体验。

我们可以使用响应式布局检测工具，输入网址 shopify.com 来进行测试，它是可以适应任何终端设备的。

我们发现，Shopify 并没有像 Amazon 一样采用复杂的页面布局，更没有放置太多的内容在首页，这使他们可以轻而易举地实现响应式布局。简洁的网站设计可以让访客以最快的速度找到自己想要的东西。

Shopify 的设计可谓是视觉效果与用户体验的完美结合，是采用全球设计趋势的典型案例，值得电商平台企业参考。

据 悉，目前 Shopify 的用户已经超过了2.5万，其惊人潜力也使它成为资本追逐的对象，Shopify 已经从Bessemer、FirstMark资本等投资公司获得了2200万美元的投资。相信在未来的发展中，我们可以欣赏到 Shopify 更多的设计风采。
