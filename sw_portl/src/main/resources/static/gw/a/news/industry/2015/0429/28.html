H5时代终将到来，微软放弃IE，为盗版用户升级

织梦科技•2015-04-29•行业动态•

被微软IE浏览器各种版本的兼容性问题，折磨了这么多年的网站制作师们，终于有盼头了，HTML5时代终将到来。我们来看近期的两则业内新闻：

1. 微软将为盗版用户免费提供 Windows10 的升级服务

据 美国科技新闻网站“连线”报道，在中国深圳举办的一次大会上，微软负责Windows操作系统业务的高管梅森，公布了有关Windows10发布时间和升 级的更多细节内容。根据微软新政策，安装盗版 Windows7 和 Windows8 的用户，也能够免费升级到Windows10。

2. Windows10 的斯巴达浏览器将逐步取代 IE

斯巴达浏览器将使用新型渲染引擎与全新用户界面，同时还与谷歌Chrome一样支持扩展插件。斯巴达浏览器将取代IE11成为Win10手机唯一浏览器，但IE11仍会在Win10 PC版系统中与斯巴达浏览器共存，因为前者能为传统网站提供更好的兼容性。

不得不说，在谷歌和苹果的逼迫之下，微软的软件业务，最终还是走上了免费增值模式。但无论如何这确实还是振奋人心的两条消息，微软的改革决心看来并不是装装样子而已。

斯 巴达浏览器 ( Spartan web browser ) 不再像IE6一样任性，它更好地响应了W3C标准，全面支持HTML5技术，为推进网页前端标准化做出了历史性的贡献。微软为盗版 Windows 用户免费升级 Windows10 的决定，在大幅提高Win10普及率的同时，无疑也为斯巴达浏览器逐步取代IE铺平了道路。可以预见不久的将来，在浏览器标准高度统一的时代，HTML5 标准将遍布整个互联网。
